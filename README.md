# Lapland Finland Travel Website

A simple, responsive travel landing page for Lapland, Finland.

## Pages

- `index.html` — Homepage with destination info, gallery, and map
- `login.html` — Login page
- `register.html` — Registration page

## Files

- `style.css` — Shared stylesheet
- `script.js` — Mobile menu toggle, gallery lightbox, and form handling

## Features

- Full-screen hero background image
- Interactive photo gallery with lightbox
- Embedded Google Maps
- Login & register forms with localStorage
- Mobile-responsive layout
